each # run block for each element of array
        arr.each {|x|puts(x)}
            1
            2
            3
            ...
        Use: Running a state operation. By default it outputs to stdOUT but with each it is redirectable, to another variable, stdOUT or otherwise. If not explicitly captured the result is lost as soon as its iterated. This never changes the original so if we store the output we can have a copy of the original and the solution.
